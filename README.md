# 🏭 Assemly-Sys (Laptop Assembly Scanner)

Sistem pelacakan produksi perakitan laptop berbasis web untuk mempercepat pencatatan komponen menggunakan barcode scanner dan mencegah duplikasi data.

## 🚀 Fitur Utama
* **Smart Barcode Recognition**: Otomatis mengenali jenis komponen (RAM, SSD, dll) dari prefix barcode.
* **Anti-Duplikasi**: Mencegah pemindaian ulang barcode yang sama.
* **Real-time Dashboard**: Visualisasi performa produksi per lini secara langsung.
* **Export Excel**: Mengunduh laporan aktivitas produksi dengan mudah.

## 🛠️ Teknologi
* Framework: Laravel 10.
* Database: MySQL.
* Frontend: Bootstrap 5 & jQuery AJAX.
